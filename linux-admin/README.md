# Linux Administration Labs

This area holds hands-on labs and configs that demonstrate practical Linux admin skills.

## Folders

- `ansible/` – Playbooks and roles.
- `bash/` – Shell scripts and one-liners.
- `python/` – Python automation tools.
- `docker/` – Container and image examples.
- `rhel/` – RHEL-specific labs and configs.
- `arch/` – Arch/learning labs.
- `configs/` – Reusable config files.
- `labs/` – Larger, multi-step lab projects.
