RHEL WSL – Podman + Ansible Webstack

Deployed automatically with Ansible onto a RHEL-on-WSL host.

Maintainer: Benjamin Panzer (sneakyturtlekungfu)

Project repo: linux-cloud-security-mastery

What this demo shows

  • Podman pod named webstack
  • Nginx front-end serving this page
  • whoami backend container sharing the same pod

Example backend check from the host:

podman exec -it webstack-web curl http://127.0.0.1
